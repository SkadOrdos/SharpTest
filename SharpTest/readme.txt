Приложение представляет из себя веб сервис который обрабатывает GET/POST запросы для получения и обновления набора чисел. В качестве хранилища разрешено использовать файл на диске вместо БД для упрощения логики. В приложении должен быть предусмотрен отдельный модуль для расчетов статистики по набору чисел отдельно от проекта сервиса. Также необходимо предусмотреть простую авторизацию (логин и пароль, например в виде Basic). Приложение должно корректно обрабатывать параллельные запросы.
Прим: (Пароль тестового пользователя "q": 1)
